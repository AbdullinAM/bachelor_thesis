\abstractcontent{
Бакалаврская работа посвящена автоматическому извлечению контрактов функций из исходного кода программ. Рассмотрены современные исследования в этой области. Сформулирована конкретная задача извлечения контрактов. Расработан алгоритм, который позволяет извлечь контракты из исходного кода. Разработанный алгоритм реализован в системе статического анализа Borealis. Проведено тестирование полученного инструмента.
}
