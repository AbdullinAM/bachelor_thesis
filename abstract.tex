\abstractcontent{
Бакалаврская работа посвящена автоматическому извлечению контрактов функций из исходного кода программ. Рассмотрены современные исследования в этой области. Сформулирована конкретная задача извлечения контрактов для повышения полноты и точности статического анализа. Разработан алгоритм, который позволяет извлечь контракты из исходного кода. Разработанный алгоритм реализован в системе статического анализа Borealis. Проведено тестирование полученного инструмента на реальных программных проекта. Тестирование показало целесообразность применения предлагаемого подхода. 

В результате работы была создана технология автоматического извлечения контрактов и разработан прототип, реализующий эту технологию.
}
